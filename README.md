# Responsive Layout Using Flexbox

## Task Description
This project demonstrates a responsive web layout created 
using CSS flexbox as part of an internship task.

## Tools Used
- HTML
- CSS3 (Flexbox)
- VS code
- Browser Developer Tools

## Features
- Responsive navigation bar
- Flexbox-based card layout
- Mobile-friendly design
- Flex-wrap for smaller screens

## Responsiveness
The layout adjusts automatically for different screen
sizes using Flexbox properties and media queries.

## Outcome
Successsfully implemented a responsive layout using
Flexbox following the given guidelines.
